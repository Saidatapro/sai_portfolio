# Netflix Style Portfolio

A premium, responsive portfolio website inspired by the Netflix UI.

## Features
- **Netflix Aesthetics**: Dark mode, red accents, carousel rows, and card hover effects.
- **Responsive Design**: Works on desktop and mobile.
- **Interactive**: Hover effects, modal previews, and smooth scrolling.
- **Vanilla Tech Stack**: HTML5, CSS3, ES6 JavaScript (No frameworks required).

## Setup
1. Simply open `index.html` in your browser.
2. OR serve with Python:
   ```bash
   python3 -m http.server 8081
   ```
   Then visit `http://localhost:8081`.

## Customization
- Edit `script.js` to update the `data` object with your real Experience, Projects, and Skills.
- Replace image URLs in `script.js` with your own hosted images.
- Modify `styles.css` to tweak colors or layout.
